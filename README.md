## Setup
To run this project, install it locally using npm:

1. clone the repository into your machine.
   ```sh
   git clone https://github.com/dhifaouibilel/TodoList-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the app
 ```sh
   nodemon app.js
   ```
